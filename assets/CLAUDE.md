# assets/

Static files served directly to the browser.

## Contents

- `josh_mayanja_resume.pdf` — resume linked from the site's navigation and contact section. Replace this file (keep the same filename) to update the resume.
- `videos/qu_game_film.mp4` — hockey game film embedded on `extras.html`.

## Notes

- The resume PDF is tracked via Git LFS (see `.gitattributes`).
- When replacing the resume, make sure the new PDF is also committed via LFS rather than as a regular binary blob.
